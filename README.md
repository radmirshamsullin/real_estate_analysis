# Анализ рынка недвижимости Ленинградской области

**Цель проекта:**  
Провести исследовательский анализ рынка недвижимости Санкт-Петербурга и Ленинградской области, оценить активность объявлений, сезонность и характеристики объектов.

**Описание проекта:**  
- Выполнен анализ времени активности объявлений и выявлены самые распространённые категории объектов  
- Оценено влияние характеристик недвижимости на активность объявлений и различия между регионами  
- Исследована сезонность публикаций и снятий объявлений  
- Проведен анализ рынка недвижимости Ленинградской области

**Навыки и технологии:**  
- SQL (PostgreSQL)  
- DBeaver  
- Анализ данных и экспорт результатов в CSV  
- Подготовка отчётов для бизнеса

**Структура репозитория:**
```
real_estate_analysis/
├── README.md           # текущий файл
├── sql_queries.sql     # основной файл с SQL-запросами
├── dashboard_link.txt  # ссылка на дашборд в Yandex DataLens
├── results/		# папка с результатами выполнения запросов
│   ├── ads_categories.csv
│   ├── ads_features_region.csv
│   ├── ads_seasonality.csv
│   └── lenoblast_market.csv
```

**Примечание по данным:**  
- В репозитории не хранятся исходные базы данных.  
- Все результаты запросов сохранены в CSV-файлах для демонстрации.
